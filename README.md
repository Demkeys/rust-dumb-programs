## Rust Dumb Programs
This is a repo of silly programs I made to practice Rust. Each of these ideas was given by Gemini. In each prompt I gave it a list of the Rust concepts I was familiar with at the time, because when starting this project I hadn't explored all Rust concepts yet, but still wanted to make practice programs. I then asked it to give me an idea for a silly program to make based on my current knowledge. These programs are just meant to be silly programs for practice purposes, not necessarily functional programs that do something meaningful.

My primary motivation for this is my philosophy that: 
> If you are learning a new programming language, make silly dumb program with it. Doesn't matter if you never public the programs. Doesn't even matter if the programs don't really do anything meaningful. Just make dumb programs, and over time, you'll gain a lot of practical knowledge and experience in the language, along with very useful problem-solving skills.

_NOTE: Keep in mind, these are meant to be silly programs that I'm writing to practice and explore Rust. The intent here isn't to write good code, it's to just write code. So don't expect good practices, good performance, etc. I'm just having fun with the language here._

### Usage
They're all part of the same project, set up as examples. So you can run example ex01 using ```cargo run --example ex01``` and so on.
