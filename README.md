## Rust Dumb Programs
This is a repo of silly programs I made to practice Rust. Each of these ideas was given by Gemini. In each prompt I gave it a list of the Rust concepts I was familiar with at the time, because when starting this project I hadn't explored all Rust concepts yet, but still wanted to make practice programs. I then asked it to give me a idea for a silly program to make based on my current knowledge.

### Usage
They're all part of the same project, set up as examples. So you can run example ex01 using ```cargo run --example ex01``` and so on.
